# Proyectos

## 1. CR0SS Chains — Tienda de Joyería
Página de e-commerce desarrollada en WooCommerce para una marca de joyería. Ofrece una experiencia de compra en línea con catálogo de productos, carrito de compras y gestión de pedidos.

🔗 [cr0sschains.com](https://cr0sschains.com/)

## 2. Legacy Museum — Sistema para Museo
Sistema web completo diseñado para la gestión de un museo. Incluye funcionalidades administrativas como control de acceso mediante inicio de sesión, manejo de colecciones y organización de información institucional.

🔗 [legacy-museum.vercel.app](https://legacy-museum.vercel.app/login)
codigo: https://github.com/disloyalcobra/LegacyMuseum

🔗 [app-medico-v2.vercel.app](https://app-medico-v2.vercel.app/)

## 4. Supply Game — Plataforma Educativa de Cadena de Suministro
Página web didáctica e interactiva enfocada en la enseñanza de la cadena de suministro. Integra un videojuego, quizzes temáticos y un sistema de rankings para hacer el aprendizaje dinámico y competitivo.

🔗 [supply-game.vercel.app](https://supply-game.vercel.app/)
codigo: https://github.com/disloyalcobra/SupplyGame
